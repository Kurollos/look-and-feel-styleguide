# Styleguide

# Styleguide Embassy Free Mind

## Kleuren
- Primary: #002646
- Secondary: #ffffff
- Accent: #d8b66a

## Lettertypes
- Body: Arial, Helvetica, sans-serif
- Headers: Cinzel

## Spacing
- --spacing-s: 15px (kleine spacing)
- --spacing-m: 30px (middelgrote spacing)

## Classes

### Knoppen
- `.button-primary` → blauwe knop met witte tekst, hover: achtergrond kleur verandert naar accentkleur
- `.button-secondary` → witte knop met blauwe rand, hover: achtergrond blauw en tekst wit

### Navigatie
- `.nav-link` → standaard link in navigatie, met svg rotatie bij hover

### Headers
- `h1` → grootste kop, Cinzel, responsive font-size
- `h2` → middelgrote kop, uppercase, gewicht 400
- `h3` → kleine kop, gewicht 400

### Tekst
- `p` → standaard paragraaf, regelhoogte 1.4, gewicht 400

## Animaties
- Hover knoppen: scale 1.05
- Hover nav-link svg: rotate 90° → -90° op hover
- Animatieduur: 0.3s (variabele: --animation-duration)
